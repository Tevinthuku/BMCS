Java is 2 things

  • A Programming and computing platform
  • A runtime environment


1. A Programming and computing Platform

A lot of apps will not run unless you have java installed. Its a whole lot simpler to use unlike other programming lang. like C. It uses an object oriented model in that we must create classes and instanciate objects from the classes.

2. A runtime environment


This means that java can also be seen as a program/ runtime environment that needs to be installed on a computer in order to run java based programs.

Differences btwn Java and Javascript


Though having almost the same name, the 2 are quite different programming languages.
Java is an Object Oriented Language whereas Javascript is an intepreted language that runs in the browser and helps in manipulating the DOM.
Java On the other hand ,is used in stand alone applications,like the browser you are viewing this page from.
